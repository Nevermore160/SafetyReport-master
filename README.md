# SafetyReport

> 基于SpringBoot的安全报告生成项目



### 技术点

1. 后端:springBoot、Mybatis、FreeMarker、PhantomJS、EasyPOI
2. 存储:Mysql、Redis
3. 前端:vue、echarts
4. 部署:Dockerfile



### 文件说明

--AHreport  	   后端项目
--Docker   		   Dockerfile
--ReporteAnalysis  前端项目
